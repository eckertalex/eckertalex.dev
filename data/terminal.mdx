## Shell
- [zsh with Oh My Zsh](https://ohmyz.sh/)
  ```bash:title=~/.zshrc
  export ZSH="/Users/<username>/.oh-my-zsh"
  ZSH_THEME="spaceship"

  VSCODE=code-insiders

  plugins=(
      git
      npm
      colored-man-pages
      colorize
      brew
      osx
      common-aliases
      history-substring-search
      zsh-completions
      zsh-autosuggestions
      zsh-syntax-highlighting
      you-should-use
      thefuck
      themes
      vscode
      wd
  )

  source $ZSH/oh-my-zsh.sh

  fpath=(/usr/local/share/zsh-completions $fpath)

  DEFAULT_USER=`whoami`

  eval "$(fnm env --multi)"

  eval $(thefuck --alias)

  autoload -Uz compinit
  compinit

  # Completion for kitty
  kitty + complete setup zsh | source /dev/stdin
  ```

### Configuration

Putting it all together. Under `~/.config/kitty/` create a `kitty.conf` and add the following:

```:title=~/.config/kitty/kitty.conf
# theme
include             night_owl.conf

# startup session
startup_session     startup_session
```

In `start_session` we can configure how many windows and tabs are open and in which layout, and much, much more.

This is my simple `startup_session`:

```:title=~/.config/kitty/startup_session
# Create media Tab
new_tab media
layout tall
# Create a window
launch

# Create dev Tab
new_tab dev
focus
layout tall
# Create a window
cd ~/dev/
launch

# Create a window
cd ~/dev/
launch
```
