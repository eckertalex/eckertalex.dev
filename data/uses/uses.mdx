## Other Apps

- [Figma](https://www.figma.com/)
  - Design App
- [Rectangle](https://rectangleapp.com/)
  - Move and resize windows in macOS using keyboard shortcuts or snap areas
- [dozer](https://github.com/Mortennn/Dozer)
  - Hide menu bar icons on macOS
- [bettertouchtool](https://folivora.ai/) -BetterTouchTool is a great, feature

  - Raycast is a blazingly fast, totally extendable launcher. It lets you

  - An open-source screen recorder built with web technology.
- [mpv](https://mpv.io/)
  - a free, open source, and cross-platform media player

  - VLC is a free and open source cross-platform multimedia player and framework

- [keepingyouawake](https://github.com/newmarcel/KeepingYouAwake)
  - Prevents your Mac from going to sleep.
- [itsycal](https://www.mowglii.com/itsycal/)
  - Itsycal is a tiny menu bar calendar.
- [Lunar](https://lunar.fyi/)
  - The defacto app for controlling monitors
- [AlDente](https://github.com/davidwernhart/AlDente)
  - MacOS menu bar tool to limit maximum charging percentage
- [Rocket](https://matthewpalmer.net/rocket/)
  - Rocket is a free Mac app that makes typing emoji faster and easier using
    Slack-style shortcuts.
- [HandMirror](https://handmirror.app/)
  - A one-click camera check, right from the menu bar
