### CLIs
- [git](https://git-scm.com/) - See my [git configuration](/git)
- [gh](https://cli.github.com/) - Official GitHub CLI tool
- [zsh-z](https://github.com/agkozak/zsh-z) - smart `cd`
- [wget](https://www.gnu.org/software/wget/)
- [fzf](https://github.com/junegunn/fzf) - a command-line fuzzy finder
- [fd](https://github.com/sharkdp/fd) - a better find
- [bat](https://github.com/sharkdp/bat) - a better cat
  - Save [night-owlish.tmTheme](https://github.com/batpigandme/night-owlish/blob/master/tmTheme/night-owlish.tmTheme) to `~/.config/bat/themes/`
  - Run `bat cache --build`
  - Edit [`~/.config/bat/config`](https://github.com/eckertalex/dotfiles/blob/main/.config/bat/config)
    ```:title=~/.config/bat/config {1-3}
    --theme="night-owlish"
    --italic-text=always
    --map-syntax "*.mdx:Markdown"
    ```
- [delta](https://github.com/dandavison/delta) - a better git diff. See my delta configuration at my [git configuration](/git#git-diff-delta-theme)
- [ripgrep](https://github.com/BurntSushi/ripgrep) - a better grep
- [fnm](https://github.com/Schniz/fnm) - a node.js version manager
- [homebrew](https://brew.sh/) (on MacOS)
- [tldr](https://tldr.sh/)
- [trash](https://github.com/sindresorhus/trash-cli)
