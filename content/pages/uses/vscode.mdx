### <a name="generate-extension-list"></a>How to generate a list of VSCode Extensions 

I just load the following snippet up in a Quokka buffer​​​​ and the list will be copied to my clipboard.

```js:title=generate-vscode-extensions-list.js
const {execSync, spawn} = require('child_process')

const result = execSync('code-insiders --list-extensions')

const list = String(result)
  .split('\n')
  .filter(Boolean)
  .map(
    x => `- [${x}](https://marketplace.visualstudio.com/items?itemName=${x})`
  )
  .join('\n')

// const proc = spawn('clip') // for Windows
const proc = spawn('pbcopy') // for MacOS
proc.stdin.write(list)
proc.stdin.end()
```
