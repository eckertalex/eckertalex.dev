## How to generate a list of VSCode Extensions

```js:generate-vscode-extensions-list.js
const {execSync, spawn} = require('child_process')

const result = execSync('code --list-extensions')

const list = String(result)
  .split('\n')
  .filter(Boolean)
  .map(
    (x) => `- [${x}](https://marketplace.visualstudio.com/items?itemName=${x})`,
  )
  .join('\n')

// const proc = spawn('clip') // for Windows
const proc = spawn('pbcopy') // for MacOS
proc.stdin.write(list)
proc.stdin.end()
```
